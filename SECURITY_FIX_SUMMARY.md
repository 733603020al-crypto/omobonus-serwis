После обновления React до 19.2.3 **обязательно** очистите кеш Next.js:

```bash

npm run dev:safe
```

Или вручную:
```powershell
Remove-Item -Recurse -Force .next
Remove-Item -Recurse -Force .turbo
npm install
npm run dev:safe
```
